Introduction

What is this?

This utility software is designed to simplify the process of building TGA files into VTF spritesheets for 64-bit Team Fortress 2 (TF2). It utilizes mksheet.exe and vtex.exe found in the bin\x64 directory, making the creation of spritesheets for TF2 more accessible.

Why?

As I updated the Transparent Flamethrower Fire mod to x64, I found it cumbersome to work with mksheet.exe and vtex.exe. Searching online led me to this repository, which still uses the original TF2's tools. I modified the paths to use the 64-bit versions and ported the project to WPF, adding several quality-of-life enhancements.

This tool fills a niche that was lacking in support for texture creation within TF2's ecosystem. I hope it proves useful to others.

Cheers!

Getting Started

1. Setup

  1. Ensure the Team Fortress 2 folder is set to the game's executable directory.
  2. Set the "Folder Containing TGA Frames" to the folder containing your frames. PNG files can also be used, and the transparency will be preserved as the alpha channel in the TGAs. You can batch convert PNG files to TGA using the provided option.
  3. Input files cannot contain spaces; however, they will be automatically renamed as mksheet.exe does not support filenames with spaces.
  4. Specify the "Input Filename Prefix" (e.g., [prefix_number]).
  5. Set the "Output Filename" for the resulting .VTF file.

Once these steps are completed, you can select additional options like "Multiple Sequences" (for multiple animations requiring manual configuration) or "Loop Animation" (which repeats the sequences).

For more information, refer to the Valve Developer Wiki.

2. Configuration

After setup, click "Generate MKS File" to create a .mks file in the "Folder Containing TGA Frames". This file can be opened directly by clicking "Open MKS".

The "Open MKS" button will prompt you to select a text editor. You can choose to use the system's default text editor or manually select one. You also have the option to disable future prompts by selecting "Don't ask again".

Once the MKS file is configured, click on "VTEX Configuration" to set up the parameters for vtex.exe. (See the VTEX Configuration Help for more details.)

3. Generation & Compiling

Click "Create VTF" to build the SpriteSheet VTF file. The output will be saved to Folder Containing TGA Frames\[output]\[filename].vtf, and the output folder will open automatically.

VTEX Config Window

This window provides a textbox with autofill suggestions and validation to ensure correct parameters. You can save configurations as templates using the dropdown menu in the top right corner.

For more detailed information on vtex.exe parameters, refer to the Valve Developer Wiki.
